# Build Together Repository Instructions

1. Clone the repository to your local machine.
2. Install the dependencies.
    ```
    npm install
    ```

3. Run the server 
    1. In development mode.
    ```
    npm run dev
    ```
    2. In production mode.
    ```
    npm start
    ```
